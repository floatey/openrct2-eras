# Era Progression System
!! THIS PLUGIN IS IN DEVELOPMENT, PLEASE BACKUP YOUR SAVES BEFORE YOU TRY IT OUT !!

OpenRCT2 plugin — progress era by era, unlock more rides by building rides.

**v0.3.3** • API 77+ • MIT

## Quick Start

1. Drop `era-progression.js` into `plugin/` folder
2. Menu → **Era Progression System** → **Initialize**
3. Build rides → unlock next era when you reach the target

## Eras

- **Antique Amusement (1890s-1910s)** — 5 unqiue rides to progress
- **Classic Coasters (1920s-1930s)** — 5 rides unqiue rides to progress
- **Transition Era (1950s)** — 6 rides unqiue rides to progress
- **Steel Renaissance (1960s-1970s)** — 6 rides unqiue rides to progress
- **Extreme Innovation (1980s)** — 8 rides unqiue rides to progress
- **Modern Thrill Revolution (1990s)** - 8 rides unqiue rides to progress
- **Millennium Age (2000s+)**

Research is **disabled**. Only era-appropriate content is available. 

Plugin can be disabled via debug menu.

All rides required for this plugin are enabled in the scenario upon initialization.
*(excluding rct2ww, rct2tt, rct1ll, rct1)
